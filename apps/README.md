# Application Submodules

Application repositories are attached here as Git submodules.

Planned paths:

- `gateway/` — Go API gateway and SSE endpoint.
- `commerce/` — Medusa.js v2 commerce engine and workers.
- `superadmin-web/` — global platform administration.
- `tenant-admin-web/` — tenant administration, OMS, and WMS UI.
- `storefront-web/` — customer storefront.
- `infrastructure/` — deployment and observability configuration.

Each release must pin every submodule to an explicit commit.
